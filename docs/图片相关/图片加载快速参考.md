# 图片加载快速参考卡片

## 🚀 5 分钟快速上手

### 步骤 1：准备图片
```bash
# 准备一张 PNG 或 JPG 图片
my_icon.png  (推荐 64x64 或 128x128)
```

### 步骤 2：转换为字节数组
```bash
python Scripts/image_to_cpp.py my_icon.png my_icon
```

### 步骤 3：添加到 image.h
```cpp
// 文件：src/Assets/image.h
unsigned char my_icon[] = {
    0x89, 0x50, 0x4E, 0x47, // ... 粘贴生成的字节数据
};
```

### 步骤 4：声明纹理指针
```cpp
// 文件：src/Hook/DX11Hook.cpp (约第 50 行)
namespace image {
    // ... 现有的声明
    ID3D11ShaderResourceView* my_icon = nullptr;  // 新增
}
```

### 步骤 5：加载纹理
```cpp
// 文件：src/Hook/DX11Hook.cpp (约第 430 行)
#ifdef USE_EMBEDDED_IMAGES
if (LoadTextureFromMemory(g_pd3dDevice, my_icon, sizeof(my_icon), &image::my_icon)) {
    printf("[+] My Icon 加载成功\n");
}
#endif
```

### 步骤 6：在 UI 中显示
```cpp
// 文件：src/UI/Menu/Menu.cpp
namespace image {
    extern ID3D11ShaderResourceView* my_icon;  // 外部声明
}

// 在渲染函数中
if (image::my_icon) {
    ImGui::GetWindowDrawList()->AddImage(
        image::my_icon,
        ImVec2(100, 100),      // 左上角
        ImVec2(164, 164),      // 右下角 (100+64)
        ImVec2(0, 0),          // UV 起点
        ImVec2(1, 1),          // UV 终点
        ImColor(255, 255, 255, 255)  // 白色 = 原图颜色
    );
}
```

## 📐 常用代码片段

### 居中显示
```cpp
ImVec2 pos = ImGui::GetWindowPos();
ImVec2 size = ImGui::GetWindowSize();
float iconSize = 64.0f;
float centerX = (size.x - iconSize) / 2.0f;
float centerY = (size.y - iconSize) / 2.0f;

ImGui::GetWindowDrawList()->AddImage(
    image::my_icon,
    pos + ImVec2(centerX, centerY),
    pos + ImVec2(centerX + iconSize, centerY + iconSize),
    ImVec2(0, 0), ImVec2(1, 1),
    ImColor(255, 255, 255, 255)
);
```

### 半透明效果
```cpp
ImColor(255, 255, 255, 128)  // 50% 透明
```

### 悬停高亮
```cpp
bool isHovered = ImGui::IsMouseHoveringRect(pos, pos + size);
ImU32 color = isHovered 
    ? ImColor(255, 255, 255, 255)  // 完全不透明
    : ImColor(255, 255, 255, 200); // 稍微透明
```

### 淡入淡出
```cpp
static float alpha = 0.0f;
alpha += ImGui::GetIO().DeltaTime * 2.0f;
if (alpha > 1.0f) alpha = 0.0f;
ImColor(255, 255, 255, (int)(alpha * 255))
```

## 🔧 常见问题速查

| 问题 | 解决方案 |
|------|---------|
| 图片不显示 | 检查 `USE_EMBEDDED_IMAGES` 是否定义 |
| 编译错误 | 确认字节数组语法正确 |
| 颜色不对 | 修改 `ImColor` 参数 |
| 位置不对 | 检查坐标计算 |
| 尺寸不对 | 修改右下角坐标 |

## ⚡ 一键复制模板

### 完整代码模板
```cpp
// ============================================================================
// 1. 在 src/Assets/image.h 中添加
// ============================================================================
unsigned char my_icon[] = {
    // ... 字节数据
};

// ============================================================================
// 2. 在 src/Hook/DX11Hook.cpp 中声明（约第 50 行）
// ============================================================================
namespace image {
    ID3D11ShaderResourceView* bg = nullptr;
    ID3D11ShaderResourceView* logo = nullptr;
    ID3D11ShaderResourceView* roll = nullptr;
    ID3D11ShaderResourceView* my_icon = nullptr;  // 新增
}

// ============================================================================
// 3. 在 src/Hook/DX11Hook.cpp 中加载（约第 430 行）
// ============================================================================
#ifdef USE_EMBEDDED_IMAGES
if (LoadTextureFromMemory(g_pd3dDevice, my_icon, sizeof(my_icon), &image::my_icon)) {
    printf("[+] My Icon 从内嵌数据加载成功\n");
} else {
    printf("[!] 错误: My Icon 内嵌数据加载失败！\n");
}
#endif

// ============================================================================
// 4. 在 src/UI/Menu/Menu.cpp 中使用
// ============================================================================
namespace image {
    extern ID3D11ShaderResourceView* bg;
    extern ID3D11ShaderResourceView* logo;
    extern ID3D11ShaderResourceView* roll;
    extern ID3D11ShaderResourceView* my_icon;  // 新增
}

void RenderMenu() {
    ImVec2 pos = ImGui::GetWindowPos();
    
    // 显示图片
    if (image::my_icon) {
        ImGui::GetWindowDrawList()->AddImage(
            image::my_icon,
            pos + ImVec2(20, 20),       // 左上角
            pos + ImVec2(84, 84),       // 右下角 (20+64)
            ImVec2(0, 0),               // UV 起点
            ImVec2(1, 1),               // UV 终点
            ImColor(255, 255, 255, 255) // 颜色
        );
    }
}
```
