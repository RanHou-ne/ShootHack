# 图片颜色原理说明

## 🎨 为什么同一张图片显示不同颜色？

你可能注意到，在 UI 中左上角和左下角的 Logo 看起来颜色不同，但它们实际上使用的是**同一个图片资源**！

## 原理解析

### ImGui 的颜色叠加功能

ImGui 的 `AddImage` 函数支持颜色叠加（Tint Color）：

```cpp
void AddImage(
    ImTextureID user_texture_id,    // 纹理指针
    const ImVec2& p_min,            // 左上角坐标
    const ImVec2& p_max,            // 右下角坐标
    const ImVec2& uv_min,           // UV 起点
    const ImVec2& uv_max,           // UV 终点
    ImU32 col                       // ← 颜色叠加参数！
);
```

最后一个参数 `col` 控制图片的颜色效果。

## 实际案例

### 代码位置：`src/UI/Menu/Menu.cpp`

#### 左上角 Logo（第 1186 行，已注释）
```cpp
ImGui::GetWindowDrawList()->AddImage(
    image::logo,                           // 纹理
    pos + ImVec2(14, 14),                  // 左上角
    pos + ImVec2(46, 46),                  // 右下角
    ImVec2(0, 0),                          // UV 起点
    ImVec2(1, 1),                          // UV 终点
    ImColor(255, 255, 255, 255)            // 白色 = 原图颜色
);
```

#### 左下角 Logo（第 1336 行）
```cpp
ImGui::GetWindowDrawList()->AddImage(
    image::logo,                           // 同一个纹理！
    pos + ImVec2(...),                     // 不同位置
    pos + ImVec2(...),
    ImVec2(0, 0),
    ImVec2(1, 1),
    ImColor(255, 255, 255, 255)            // 白色 = 原图颜色
);
```

**关键点：两处代码使用的是同一个 `image::logo` 纹理！**

## 颜色叠加规则

### 基本规则

颜色叠加采用**乘法混合**：

```
最终颜色 = 原图颜色 × 叠加颜色 / 255
```

### 常见效果对照表

| 原图颜色 | 叠加颜色 | 最终效果 | 说明 |
|---------|---------|---------|------|
| 橙色 (255, 165, 0) | 白色 (255, 255, 255) | 橙色 | 保持原样 |
| 橙色 (255, 165, 0) | 红色 (255, 0, 0) | 深红色 | 橙色×红色 |
| 白色 (255, 255, 255) | 蓝色 (0, 0, 255) | 蓝色 | 白色可变任意色 |
| 任意颜色 | 半透明白色 (255, 255, 255, 128) | 半透明原图 | 控制透明度 |
| 任意颜色 | 灰色 (128, 128, 128, 255) | 变暗 50% | 降低亮度 |

### 实用示例

```cpp
// 1. 保持原图颜色（最常用）
ImColor(255, 255, 255, 255)

// 2. 半透明效果
ImColor(255, 255, 255, 128)  // 50% 透明

// 3. 红色调
ImColor(255, 100, 100, 255)

// 4. 蓝色调
ImColor(100, 100, 255, 255)

// 5. 灰度效果
ImColor(128, 128, 128, 255)

// 6. 完全透明（不可见）
ImColor(255, 255, 255, 0)
```

## 为什么看起来颜色不同？

### 可能的原因

1. **原图本身有颜色**
   - 如果 `image::logo` 原图是橙色，使用白色叠加会保持橙色
   - 如果原图是白色，可以通过叠加任意颜色来改变显示颜色

2. **背景影响**
   - 不同位置的背景颜色不同，会影响视觉效果
   - 透明图片在不同背景上看起来颜色不同

3. **光照/阴影效果**
   - UI 可能有渐变背景或阴影效果
   - 这些效果会影响图片的视觉呈现

## 如何改变图片颜色

### 方法 1：修改叠加颜色

```cpp
// 原代码（白色 = 原图颜色）
ImGui::GetWindowDrawList()->AddImage(
    image::logo,
    pos + ImVec2(14, 14),
    pos + ImVec2(46, 46),
    ImVec2(0, 0),
    ImVec2(1, 1),
    ImColor(255, 255, 255, 255)  // ← 修改这里
);

// 改为红色调
ImColor(255, 100, 100, 255)

// 改为蓝色调
ImColor(100, 100, 255, 255)

// 改为半透明
ImColor(255, 255, 255, 128)
```

### 方法 2：使用不同的原图

如果需要完全不同的颜色，建议准备不同颜色的原图：

```cpp
// 橙色 Logo
unsigned char logo_orange[] = { /* ... */ };

// 蓝色 Logo
unsigned char logo_blue[] = { /* ... */ };

// 加载不同的纹理
LoadTextureFromMemory(g_pd3dDevice, logo_orange, sizeof(logo_orange), &image::logo_orange);
LoadTextureFromMemory(g_pd3dDevice, logo_blue, sizeof(logo_blue), &image::logo_blue);
```

## 实战技巧

### 1. 悬停高亮效果

```cpp
bool isHovered = ImGui::IsMouseHoveringRect(pos, pos + size);

ImU32 color = isHovered 
    ? ImColor(255, 255, 255, 255)  // 悬停：完全不透明
    : ImColor(255, 255, 255, 200); // 正常：稍微透明

draw_list->AddImage(image::logo, pos, pos + size, 
    ImVec2(0, 0), ImVec2(1, 1), color);
```

### 2. 淡入淡出动画

```cpp
static float alpha = 0.0f;
alpha += ImGui::GetIO().DeltaTime * 2.0f;  // 每秒增加 2.0
if (alpha > 1.0f) alpha = 0.0f;            // 循环

draw_list->AddImage(image::logo, pos, pos + size,
    ImVec2(0, 0), ImVec2(1, 1),
    ImColor(255, 255, 255, (int)(alpha * 255)));
```

### 3. 根据状态改变颜色

```cpp
ImU32 color;
if (isActive)
    color = ImColor(100, 255, 100, 255);  // 绿色 = 激活
else if (isWarning)
    color = ImColor(255, 200, 100, 255);  // 橙色 = 警告
else
    color = ImColor(255, 255, 255, 255);  // 白色 = 正常

draw_list->AddImage(image::logo, pos, pos + size,
    ImVec2(0, 0), ImVec2(1, 1), color);
```

## 调试技巧

### 如何确认原图颜色

1. **使用纯白色叠加**
   ```cpp
   ImColor(255, 255, 255, 255)  // 显示原图真实颜色
   ```

2. **在纯色背景上测试**
   ```cpp
   // 先绘制纯黑背景
   draw_list->AddRectFilled(pos, pos + size, ImColor(0, 0, 0, 255));
   // 再绘制图片
   draw_list->AddImage(image::logo, pos, pos + size, 
       ImVec2(0, 0), ImVec2(1, 1), ImColor(255, 255, 255, 255));
   ```

3. **导出原图查看**
   - 使用图片查看器打开 `image.h` 对应的原始 PNG 文件
   - 确认原图的真实颜色

## 总结

- ✅ 同一张图片可以通过颜色叠加显示不同效果
- ✅ 白色叠加 (255, 255, 255, 255) = 保持原图颜色
- ✅ 透明度通过 Alpha 通道（第 4 个参数）控制
- ✅ 颜色叠加采用乘法混合
- ✅ 可以用于实现悬停、动画、状态指示等效果
