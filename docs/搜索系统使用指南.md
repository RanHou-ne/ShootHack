# 菜单搜索系统使用指南

## 功能概述

菜单搜索系统是一个通用的功能查找工具，可以快速定位和跳转到任何菜单功能项。支持中文、英文、关键词的模糊搜索，大大提升了用户体验。

## 功能特性

### 1. 智能搜索
- **多语言支持**：同时支持中文和英文搜索
- **关键词匹配**：支持多个关键词匹配
- **模糊搜索**：不区分大小写，部分匹配即可
- **智能排序**：按相关度自动排序结果

### 2. 实时反馈
- **即时搜索**：输入时立即显示结果
- **下拉显示**：搜索结果以下拉框形式展示
- **高亮显示**：清晰展示功能名称和描述

### 3. 快速跳转
- **一键跳转**：点击搜索结果直接跳转到对应 Tab 页
- **自动关闭**：跳转后自动清空搜索框和结果

## 使用方法

### 基本使用

1. **打开菜单**
   - 按 `INSERT` 键打开菜单

2. **输入搜索关键词**
   - 在顶部搜索框中输入关键词
   - 支持中文、英文、拼音首字母

3. **查看搜索结果**
   - 搜索结果会自动显示在搜索框下方
   - 最多显示 8 条最相关的结果

4. **跳转到功能**
   - 点击任意搜索结果
   - 自动跳转到对应的 Tab 页

## 技术实现

### 架构设计

搜索系统采用模块化设计，核心组件包括：

#### 1. 搜索数据库（`MenuSearch.h`）
```cpp
struct SearchableItem
{
    const char* displayName;    // 显示名称（中文）
    const char* englishName;    // 英文名称
    const char* keywords;       // 关键词
    const char* description;    // 功能描述
    int         tabIndex;       // 所属 Tab 页
    const char* category;       // 分类
};
```

#### 2. 搜索算法
采用多维度加权匹配算法：
- **显示名称匹配**：权重 100（最高）
- **英文名称匹配**：权重 80
- **关键词匹配**：权重 50
- **描述匹配**：权重 30
- **分类匹配**：权重 10（最低）

#### 3. 结果排序
按匹配分数降序排序，确保最相关的结果排在前面。

### 核心函数

#### Search() - 执行搜索
```cpp
int Search(const char* query, 
           std::vector<SearchResult>& results, 
           int maxResults = 10);
```

**参数**：
- `query`：搜索关键词
- `results`：输出搜索结果（按相关度排序）
- `maxResults`：最多返回多少条结果

**返回值**：
- 找到的结果数量

#### ContainsIgnoreCase() - 不区分大小写匹配
```cpp
bool ContainsIgnoreCase(const std::string& haystack, 
                        const std::string& needle);
```

**功能**：
- 检查字符串是否包含子串
- 不区分大小写
- 支持中英文混合

## 性能优化

### 当前性能特点
- **搜索速度**：O(n)，n 为数据库条目数
- **内存占用**：静态数组，无动态分配
- **实时性**：输入即搜索，无延迟

### 优化建议

如果数据库条目超过 1000 条，可以考虑：

1. **使用哈希表索引**
   - 为关键词建立哈希索引
   - 加速关键词查找

2. **实现增量搜索**
   - 缓存上一次搜索结果
   - 在缓存基础上进行增量搜索

3. **异步搜索**
   - 将搜索放到后台线程
   - 避免阻塞 UI 渲染

## 常见问题

### Q: 搜索结果为什么没有显示？
A: 检查以下几点：
1. 搜索框是否有输入内容
2. 关键词是否匹配任何功能项
3. 搜索结果窗口是否被其他窗口遮挡

### Q: 如何关闭搜索结果窗口？
A: 有三种方式：
1. 点击任意搜索结果（自动关闭）
2. 清空搜索框内容
3. 点击搜索结果窗口外的任意位置

### Q: 搜索支持正则表达式吗？
A: 当前版本不支持正则表达式，仅支持简单的子串匹配。如需正则表达式支持，可以修改 `ContainsIgnoreCase()` 函数。

### Q: 可以搜索配置项的值吗？
A: 当前版本仅搜索功能名称和描述，不搜索配置值。如需此功能，需要扩展搜索数据库。

### Q: 搜索是否区分中英文？
A: 搜索同时支持中英文，不区分大小写。中文搜索区分简繁体。
