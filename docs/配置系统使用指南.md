## 配置系统概述

### 什么是配置系统？
配置系统允许你保存当前的所有设置（包括功能开关、数值参数、颜色等），并在需要时快速加载，实现不同场景的快速切换。

### 核心功能
- ✅ **保存配置**：将当前所有设置保存到JSON文件
- ✅ **加载配置**：从JSON文件恢复设置
- ✅ **自动加载**：启动时自动加载上次配置
- ✅ **配置管理**：创建、删除、检查配置文件

### 配置文件格式
配置文件使用 **JSON** 格式存储，易于阅读和编辑。

## 保存与加载

### 保存配置

#### 方法1：通过菜单保存
```cpp
// 在菜单中点击"保存配置"按钮
// 输入配置文件名，例如："近战配置.json"
```

#### 方法2：代码调用
```cpp
#include "Config.h"

// 保存当前配置
SaveConfig("我的配置.json");
```

#### 保存流程
```
1. 收集所有Options结构体中的设置
   ↓
2. 转换为JSON格式
   ↓
3. 写入文件（带格式化，4空格缩进）
   ↓
4. 显示保存成功提示
```

### 加载配置

#### 方法1：通过菜单加载
```cpp
// 在菜单中点击"加载配置"按钮
// 选择要加载的配置文件
```

#### 方法2：代码调用
```cpp
#include "Config.h"

// 加载指定配置
LoadConfig("我的配置.json");
```

#### 加载流程
```
1. 打开JSON配置文件
   ↓
2. 解析JSON数据
   ↓
3. 将数据写入Options结构体
   ↓
4. 应用所有设置
   ↓
5. 显示加载成功提示
```

## 自动配置功能

### 什么是自动配置？
自动配置功能允许程序在启动时自动加载上次使用的配置，无需手动操作。

### 自动配置文件
```json
{
    "Is_AtuoConfig": true,        // 是否启用自动配置
    "Is_First_Auto_check": false  // 是否首次检查
}
```

### 启用自动配置

#### 方法1：菜单设置
```cpp
ImGui::Checkbox("自动加载配置", &Options.Auto_Read_Config);
```

#### 方法2：代码设置
```cpp
// 保存自动配置设置
saveConfigAuto("auto_config.json", true, false);
```

### 自动配置工作流程

```
程序启动
   ↓
检查 auto_config.json
   ↓
Is_AtuoConfig == true?
   ↓ 是
加载上次配置文件
   ↓
应用所有设置
   ↓
进入游戏
```

### 自动配置函数

#### 加载自动配置
```cpp
bool loadConfigAuto(const std::string& filename, 
                    bool& isAutoConfig, 
                    bool& isFirstAutoCheck) {
    std::ifstream file(filename);
    if (!file) {
        std::cout << "无法打开配置文件" << std::endl;
        return false;
    }
    
    nlohmann::json j;
    try {
        file >> j;
    } catch (const std::exception& e) {
        std::cout << "解析JSON配置文件出错: " << e.what() << std::endl;
        return false;
    }
    
    if (j.find("Is_AtuoConfig") != j.end() && isFirstAutoCheck) {
        isAutoConfig = j["Is_AtuoConfig"].get<bool>();
    }
    
    if (j.find("Is_First_Auto_check") != j.end()) {
        isFirstAutoCheck = j["Is_First_Auto_check"].get<bool>();
    }
    
    file.close();
    return true;
}
```

#### 保存自动配置
```cpp
void saveConfigAuto(const std::string& filename, 
                    bool isAutoConfig, 
                    bool isFirstAutoCheck) {
    nlohmann::json j;
    std::ofstream file(filename);
    
    if (!file) {
        std::cout << "无法创建配置文件" << std::endl;
        return;
    }
    
    j["Is_AtuoConfig"] = isAutoConfig;
    j["Is_First_Auto_check"] = isFirstAutoCheck;
    
    file << j.dump(4);
    file.close();
}
```

## 高级用法

### 1. 配置文件管理类

```cpp
class FileHandler {
public:
    enum class FileStatus {
        NOT_EXISTS,                     // 文件不存在
        EMPTY,                          // 文件为空
        NON_EMPTY_DELETED_SUCCESSFULLY, // 非空文件删除成功
        NON_EMPTY_DELETION_FAILED       // 非空文件删除失败
    };

    static FileStatus RemoveFiile(const std::string& filename);
};
```

#### 使用示例

```cpp
// 删除配置文件
auto status = FileHandler::RemoveFiile("旧配置.json");

switch (status) {
    case FileStatus::NOT_EXISTS:
        std::cout << "文件不存在" << std::endl;
        break;
    case FileStatus::EMPTY:
        std::cout << "文件为空" << std::endl;
        break;
    case FileStatus::NON_EMPTY_DELETED_SUCCESSFULLY:
        std::cout << "删除成功" << std::endl;
        break;
    case FileStatus::NON_EMPTY_DELETION_FAILED:
        std::cout << "删除失败" << std::endl;
        break;
}
```

### 4. 配置备份

```cpp
// 备份当前配置
void BackupConfig(const std::string& original) {
    // 生成备份文件名（带时间戳）
    auto now = std::time(nullptr);
    std::string backup = original + ".backup." + std::to_string(now);
    
    // 复制文件
    std::ifstream src(original, std::ios::binary);
    std::ofstream dst(backup, std::ios::binary);
    dst << src.rdbuf();
    
    std::cout << "配置已备份到: " << backup << std::endl;
}
```

### 5. 配置对比

```cpp
// 对比两个配置文件的差异
void CompareConfigs(const std::string& config1, 
                    const std::string& config2) {
    std::ifstream file1(config1);
    std::ifstream file2(config2);
    
    nlohmann::json j1, j2;
    file1 >> j1;
    file2 >> j2;
    
    // 对比差异
    for (auto& [key, value] : j1.items()) {
        if (j2[key] != value) {
            std::cout << key << ": " 
                      << value << " -> " << j2[key] << std::endl;
        }
    }
}
```

// 加载前验证配置完整性
bool ValidateConfig(const std::string& filename) {
    std::ifstream file(filename);
    nlohmann::json j;
    
    try {
        file >> j;
    } catch (...) {
        return false;
    }
    
    // 检查必要字段
    std::vector<std::string> required = {
        "OpenAimbot", "Aim_Range", "threshold"
    };
    
    for (const auto& field : required) {
        if (j.find(field) == j.end()) {
            std::cout << "缺少字段: " << field << std::endl;
            return false;
        }
    }
    
    return true;
}

## 故障排除

### 问题1：配置无法保存
**可能原因：**
- 文件路径不存在
- 没有写入权限
- 磁盘空间不足

**解决方法：**
```cpp
// 检查并创建目录
std::filesystem::create_directories("configs/");
SaveConfig("configs/我的配置.json");
```

### 问题2：配置加载后无效
**可能原因：**
- JSON格式错误
- 字段名称不匹配
- 数据类型错误

**解决方法：**
```cpp
// 使用try-catch捕获错误
try {
    LoadConfig("配置.json");
} catch (const std::exception& e) {
    std::cout << "加载失败: " << e.what() << std::endl;
    // 加载默认配置
    LoadConfig("default.json");
}
```

### 问题3：自动配置不工作
**可能原因：**
- auto_config.json不存在
- Is_AtuoConfig设置为false

**解决方法：**
```cpp
// 重新创建自动配置文件
saveConfigAuto("auto_config.json", true, false);
```
